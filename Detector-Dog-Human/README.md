# Dog vs Human Detector

A real-time webcam classifier that detects whether it sees a **human** or a **dog** using a Teachable Machine model.

## How It Works

- Uses **TensorFlow.js** and **Google Teachable Machine** to run an image classification model directly in the browser.
- Captures webcam frames and displays live prediction confidence bars for each class.

## Usage

1. Open `index.html` in a browser.
2. Click "Allow Camera" to start the webcam.
3. The model will classify the feed in real time and show confidence percentages.

## Tech Stack

- TensorFlow.js
- Teachable Machine (Image)
- Vanilla HTML/CSS/JS
